ReactJS Interview Guide

From Virtual DOM to Advanced Hooks

Core Concepts

Virtual DOM

Це копія реального DOM, яка зберігається в пам'яті. React використовує її для оптимізації оновлень.

  • Diffing: Порівняння нової версії VDOM зі старою.
  • Reconciliation: Процес оновлення реального DOM лише в тих місцях, де сталися зміни.

JSX

Розширення синтаксису JavaScript, яке дозволяє писати HTML-подібний код всередині JS.

const element = 
Hello, {name}
;

Браузер не розуміє JSX, тому він транспілюється в React.createElement().

React Hooks

useState

Дозволяє додавати стан до функціональних компонентів.

const [count, setCount] = useState(0);
// Оновлення: setCount(prev => prev + 1);
useEffect

Обробка побічних ефектів (запити, підписки, маніпуляції з DOM).

useEffect(() => {
    console.log("Mounted");
    return () => console.log("Unmounted");
}, [dependency]);
Optimization: useMemo & useCallback
useMemo

Кешує результат обчислень.

const result = useMemo(() => slowFunc(a), [a]);
useCallback

Кешує саму функцію (щоб не створювати її на кожен рендер).

const handler = useCallback(() => { ... }, []);

Components & Props

Props (Властивості)

Дані, що передаються від батька до дитини. Вони Read-Only.

Keys

Спеціальний атрибут для списків. Допомагає React ідентифікувати, які елементи змінилися/додалися/видалилися.

Ніколи не використовуйте Index як Key, якщо список може змінюватися!
